## 创建目录

1. 在 `个人文件夹` 或者 `共享文件夹中`，分别创建好 `/moviepilot`、`/qbittorrent` 和 `/share` 这 3 个文件夹。
2. 在 `/moviepilot` 中创建 `/config` 和 `/core` 文件夹作为 MoviePilot 的目录映射。
3. 在 `/share` 中创建 `/downloads` 和 `/media`，前者作为下载目录，后者为刮削整理后的媒体库目录。

```
目录结构如下：

个人文件夹
|__ moviepilot
    |__ config
    |__ core
|__ qbittorrent
|__ share
    |__ downloads
    |__ media
```
